Using Variable Params

19 Dec 2015 on tech

Today just put together a test case to show usage of functions with variable number of parameters. 

 1
 2using System;
 3using System.Threading;
 4
 5namespace TestVariableParams
 6{
 7    class Tester
 8    {
 9        // use object[] to allow for arguments of any type
10        static void LogMsg(string msg, params object[] args)
11        {
12            Console.Out.Write("{0} ThreadId={1} ",
13                DateTime.Now.ToString("HH:MM:ss.fff"),
14                Thread.CurrentThread.ManagedThreadId);
15            Console.Out.WriteLine(msg, args);
16        }
17
18        static void Main(string[] args)
19        {
20            LogMsg("Test {0}",
21                "first arg");
22            LogMsg("This is a test {0} {1}",
23                "first arg",
24                "second arg");
25            LogMsg("This is a test with int args {0} {1}",
26                123,
27                456);
28        }
29    }
30}
31

The output shows- 

10:05:04.208 ThreadId=1 Test first arg
10:05:04.244 ThreadId=1 This is a test first arg second arg
10:05:04.244 ThreadId=1 This is a test with int args 123 456
