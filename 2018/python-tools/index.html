Python's Data Science Toolkit

Essentially, there are five tools in Python’s data science toolkit. These are NumPy, SciPy, Pandas, MatplotLib, and JupyterLib. I’ll jot down the essential notes on each of these toolkits.

NumPy

Why use NumPy arrays?

NumPy’s main benefit is its array object called ndarray. Ndarray objects wraps around multi-dimensional data- and allows for

  • many creational patterns- from other lists, from csvs,
  • fast matrix operations without for-each loops
  • fast data element access as well as data splicing per column or row

Why are NumPy arrays fast?

NumPy arrays are typed arrays of a fixed size- wheras Python lists are variable data structures of varying types. NumPy arrays’ set type allows it to preallocate continuous blocks of memory and allow optimised C API to make array operations extremely quick.

Operations

NumPy 2D arrays makes holding all data into memory and running operations on them extremely easy. With matrix operations, there is no need to write for-each loops.

import numpy as np

a = np.array([5,6,7,8])
print(a * 3)
--
[15 18 21 24]
